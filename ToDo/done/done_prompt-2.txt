# 修正内容
メール送信処理のファイル添付方法を以下のとおり変更したい。 
- 原稿のショートカットキープッシュ、フォルダーパス入力、待機画像認識、ファイル名欄にファイル名入力、操作をやめる。
- 代わりに、Pythonスクリプトで「PDFフォルダ」の中から、直前にダウンロードされた該当のPDFおよびスクショ画像ファイルをメールフォーマットへ貼り付ける。
- サーチ方法は同フォルダの中からファイル名を基にサーチすること。
- 上記の通り処理させるスクリプトは、別で添付されている「gpt log1.txt」を参考にして生成すること。
