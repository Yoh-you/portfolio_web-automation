1. フォルダ内を検索
import os

target = "invoice_2025.xlsx"
base_dir = r"C:\Users\YuHya\Documents\invoices"

found_path = None
for root, dirs, files in os.walk(base_dir):
    if target in files:
        found_path = os.path.join(root, target)
        break
2. Outlookメールを自動生成（添付 or 本文に貼る）
Outlookは win32com.client を使えば完全に操作できる。

添付ファイルとして追加
import win32com.client as win32

outlook = win32com.client.Dispatch("Outlook.Application")
mail = outlook.CreateItem(0)
mail.To = "example@example.com"
mail.Subject = "自動送信テスト"
mail.Body = "指定のファイルを送ります。"

mail.Attachments.Add(found_path)

mail.Display()  # ←送信前に画面で確認したい場合
# mail.Send()  # ←そのまま送るならこれ
