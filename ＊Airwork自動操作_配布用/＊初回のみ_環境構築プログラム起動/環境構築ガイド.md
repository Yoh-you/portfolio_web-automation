# 🔧 初回環境構築ガイド

## 📋 概要

このプログラムを初めて使う方は、以下の手順でPython環境をセットアップしてください。

**所要時間**: 約5〜10分（セットアップ時間） + 事前準備

## 📦 ファイルの確認

### 必要なファイルのチェック

このフォルダ内に以下のファイルが揃っていることを確認してください：

- ✅ `python-3.12.10-amd64.exe` （Pythonインストーラー）
- ✅ `右クリック→「Powershellで実行」で起動.ps1` （自動セットアップスクリプト）
- ✅ `requirements.txt` （必要なパッケージ一覧）

> **注意**: これらのファイルは配布パッケージに同梱されています。不足している場合は、管理者に連絡してください。

## 🚀 環境構築の手順

### ステップ1: 自動セットアップの開始

1. **`右クリック→「Powershellで実行」で起動.ps1`** を右クリック

2. **「PowerShellで実行」を選択**

   > ⚠️ **「PowerShellで実行」が表示されない場合**:
   > - ファイルを右クリック → 「プログラムから開く」
   > - 「Windows PowerShell」を選択

3. **セキュリティ警告が表示された場合は「実行」をクリック**

### ステップ2: セットアップの実行

1. 以下のメッセージが表示されます：
   ```
   Pythonをインストールしてもよろしいですか？ (y/n)
   ```

2. **`y`** を入力して **Enter** を押してください

3. **自動セットアップが開始されます**（5〜10分かかります）

   処理内容：
   - Python 3.12.10 のインストール
   - 仮想環境の作成
   - 必要なパッケージのインストール

### ステップ3: セットアップ完了

1. 「**セットアップが完了しました！**」と表示されたら **Enter** を押して終了

## ⚙️ プログラム実行前の準備

### A. ダウンロードフォルダの作成

以下のフォルダを作成してください：
```
C:\Users\[あなたのユーザー名]\Downloads\pdf
```

### B. Microsoft Edge の設定

1. **Edge ブラウザを開く**
2. **設定**（右上の「…」）→ **「ダウンロード」**
3. 以下を設定：
   - **ダウンロード先**: `C:\Users\[ユーザー名]\Downloads\pdf`
   - **「ダウンロード時に保存先を確認する」**: **OFF**

### C. Microsoft Outlook の起動

プログラム実行前に **Outlook を起動** しておいてください。

## ✅ セットアップ完了！

これで初回セットアップは完了です！

**次回からは、以下のファイルをダブルクリックするだけでプログラムを実行できます：**

📁 **親フォルダに戻って** → `処理開始_escキーで停止可能.bat`

## 🔧 トラブルシューティング

### 「Pythonインストーラーが見つかりません」

**原因**: `python-3.12.10-amd64.exe` がこのフォルダに配置されていない

**対処法**:
1. 配布パッケージの内容を確認し、インストーラーファイルが含まれているかチェック
2. 不足している場合は、管理者に連絡してファイルを入手
3. このフォルダ（`初回のみ_環境構築プログラム起動`）内に配置
4. 再度セットアップを実行

### 「PowerShellで実行できません」

**原因**: PowerShell の実行ポリシーが制限されている

**対処法**:
1. PowerShell を**管理者として**起動
2. 以下のコマンドを実行：
   ```powershell
   Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser
   ```
3. 再度セットアップを実行

### その他の問題

詳細なトラブルシューティングは、親フォルダの **`README_(配布用の取説).md`** をご覧ください。
